# 🚀 DryShop — Modern E-Commerce Platform

> A powerful and scalable e-commerce web application built for modern businesses.

---

## 💼 About This Project
DryShop adalah aplikasi e-commerce fullstack yang dirancang untuk memberikan pengalaman belanja online yang cepat, modern, dan user-friendly.

Cocok untuk:
- Toko online UMKM
- Brand fashion
- Digital product store
- Custom e-commerce solution

---

## ⚡ Key Features

### 🛍️ Customer Side
- Product listing & detail
- Add to cart system
- Smooth checkout experience
- Authentication (Login & Register)

### 👨‍💼 Admin Dashboard
- Dashboard overview
- Manage products & categories
- Order management
- User management
- Stock control

---

## 🧠 Tech Stack
- Laravel (Backend)
- React (Frontend)
- Inertia.js (SPA bridge)
- Tailwind CSS

## ⚙️ Installation

```bash
git clone https://github.com/Dryex-yo/Ecommerce-app.git
cd Ecommerce-app

composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed

npm install
npm run dev

php artisan serve
